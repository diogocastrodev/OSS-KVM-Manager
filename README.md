# KVM Manager

> [!NOTE]
> This Project is being developed in the purpose of a University Project

### Purpose of the project:

- Create Virtual Machines (Guest Machines)
  - Parameters: vCPU, RAM, Disk, etc...
- Power Up / Power Down
- Change Settings
- Check the VM status
- Remote Console

### Future of the Project:

- Multiple Servers
- Change Virtual Machines between Servers
